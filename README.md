I created this Blender extension primarily to automate tasks and simplify uploading models for sale using the PBR workflow from Blender's Principled BSDF shader. It also automates exporting to various 3D formats and creating textures in the file location, making it easier to upload textures to websites. Currently, it uses the F5 key to add a menu, but this will be customizable in the future.

Additionally, the extension's capabilities are also accessible through the N menu in the 3D Viewport. Some of these options only appear if the render engine is Cycles.
